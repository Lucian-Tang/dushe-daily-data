# 毒舌日报 - 微信小程序 Vibe Coding Demo

## 定位
AI 毒舌日报小程序 — "你的 AI 嘴替，每天用毒舌帮你读懂世界"

## MVP 功能（2-3天）

### 首页
- 三栏卡片：行业日报 / 开发者日报 / 社会新闻
- 每条新闻配一句毒舌点评（Lucia 人设）
- 下拉刷新

### 情绪雷达
- 今日情绪五维图：愤怒/娱乐/讨论/焦虑/正能量
- Canvas 绘制雷达图

### 分享卡片
- 点击"分享今日最毒的一句"
- 生成海报图（标题+毒舌金句+二维码）

### 历史
- 轻量级历史回顾（最近7天）

## 技术栈
- 前端：微信小程序原生 / Taro
- 后端：已有日报数据 + Supabase/CloudBase
- AI：DeepSeek V4 Pro 生成毒舌点评
- 数据：本地 JSON + 云函数

## 数据流
```
Cron 38;00 → 日报 JSON 生成 → 推送到云存储
小程序打开 → 拉取当日 JSON → AI 毒舌点评 → 展示
分享卡片 → Canvas 渲染 → 保存/分享
```

## 关键指标
- 每天 3 份日报 × 3 类 = 9 条推送
- 每条毒舌点评 ~50 tokens
- 日 API 费用 ~$0.0005
