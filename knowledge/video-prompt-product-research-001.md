## 二、分镜提示词核心要素

### 2.1 分镜结构：景别分配节奏

| 景别 | 用途 | 典型时长 | 占比建议 |
|------|------|---------|---------|
| 远景(Establishing) | 建立场景、交代环境 | 3-8秒 | 10-15% |
| 全景 | 展示整体、人物关系 | 3-6秒 | 10-15% |
| 中景 | 叙事主体、自然对话 | 4-10秒 | 30-40% |
| 近景 | 强调细节、情绪 | 2-5秒 | 20-25% |
| 特写 | 冲击感、高光时刻 | 1-3秒 | 10-15% |

**黄金节奏公式：**
```
远景(3s) → 中景(6s) → 近景(3s) → 特写(2s) → 中景(5s) → 近景(3s) → 远景(3s)
总时长约 25 秒一个完整节拍
```

### 2.2 画面描述风格

**四维描述法：**
| 维度 | 关键词示例 | 作用 |
|------|----------|------|
| 视觉风格 | 电影感、赛博朋克、水墨风、扁平插画、写实摄影 | 定调 |
| 色调 | 暖色调、冷色调、低饱和度、HDR高动态、莫兰迪色 | 情绪 |
| 氛围 | 压抑、欢快、神秘、史诗、纪实、梦幻 | 沉浸感 |
| 光线 | 逆光、侧逆光、柔光、硬光、自然光、戏剧光 | 质感 |

**优秀示例：**
- ❌ 差："一个人走在街上"
- ✅ 好："黄昏的香港街头，侧逆光勾勒人物轮廓，赛博朋克霓虹色调，写实摄影风格，柔光傍晚氛围"

### 2.3 运镜指令

| 运镜类型 | 指令关键词 | 效果 |
|---------|----------|------|
| 推 | push in, zoom in, focus on | 聚焦、强调 |
| 拉 | pull out, zoom out, wide shot | 交代环境、释放情绪 |
| 摇 | pan left/right, tilt up/down | 扫描、展示 |
| 移 | dolly left, tracking shot | 跟随、探索 |
| 跟 | follow shot, tracking | 保持主体、跟随运动 |
| 固定 | static shot, locked frame | 稳定、强调 |
| 环绕 | orbit, 360°环绕 | 强调主体、营造氛围 |
| 俯冲 | drone shot, top-down | 宏观视角、震撼感 |

### 2.4 转场方式

| 转场类型 | 触发词 | 适用场景 |
|---------|------|---------|
| 硬切 | cut to, direct cut, straight cut | 快节奏、新闻感 |
| 淡入淡出 | fade in/out, cross dissolve | 柔和过渡、情绪延续 |
| 叠化 | dissolve, overlap dissolve | 时间流逝、梦境感 |
| 滑入 | slide in, wipe | 信息对比、对比转场 |
| 模糊转场 | blur transition | 场景转换、自然过渡 |
| 黑场 | fade to black | 章节分割、休息节点 |

### 2.5 时长控制

**平台适配参考：**

| 平台 | 视频长度 | 单分镜建议 | 分镜数参考 |
|------|---------|-----------|-----------|
| 抖音/快手 | 15-60秒 | 2-5秒 | 8-15个 |
| B站/YouTube | 3-10分钟 | 3-8秒 | 25-50个 |
| 品牌广告 | 30-180秒 | 2-6秒 | 15-40个 |
| 电影级 | 2小时+ | 3-10秒 | 200+个 |

**节奏公式：**
- 开头 3 秒必须抓人（强视觉冲击或悬念）
- 每 8-12 秒出现一个小高潮或转折
- 结尾 2-3 秒留白或金句

## 三、平台爆款案例分析

### 3.1 可灵（Kling AI）

**平台特点：**
- 支持最长3分钟视频生成
- 以文生视频和图生视频为主
- 强调物理规律理解和运动自然度

**爆款Prompt结构分析：**
```
[主体] + [动作/变化] + [环境] + [风格/氛围] + [运镜]
```

**示例（仿写）：**
> "一名穿着汉服的女子在樱花树下起舞，动作流畅优雅，背景是日本京都古建筑群，电影感色调，夕阳侧逆光，轨道跟随拍摄，cinematic, 4K, 90fps"

**有效性要素：**
- 动作描述具体（"起舞"而非"动"）
- 环境细节丰富（樱花、京都、夕阳）
- 风格标签明确（cinematic, 4K）

### 3.2 即梦（Jimeng）

**平台特点：**
- 字节出品，与剪映深度整合
- 中文理解能力强
- 支持中文提示词

**爆款Prompt结构：**
```
[场景] + [主体] + [状态/动作] + [风格] + [镜头语言] + [转场]
```

**示例（仿写）：**
> "未来城市夜景，霓虹灯光反射在湿漉漉的街道上，一辆摩托车飞驰而过，电影感，俯冲航拍视角，fade in from black, 90s"

### 3.3 申途/其他平台

**通用爆款规律：**
1. 具体 > 模糊（"红色跑车" > "车"）
2. 动作 > 状态（"穿越" > "存在"）
3. 细节 > 笼统（"玻璃窗上的雨滴" > "下雨"）
4. 风格标签前置或后置都行，但要统一

## 四、标准化分镜提示词模板

### 4.1 完整分镜提示词结构

```markdown
## 分镜 [N]

【镜头】[景别] [运镜方式]
【主体】[主体描述，包括服装、动作、表情]
【环境】[背景、道具、光线、天气]
【风格】[视觉风格、色调、氛围]
【时长】[X秒]
【转场】[to next / hold / fade]
```

### 4.2 字段填写指南

| 字段 | 必填 | 填写要点 | 示例 |
|------|------|---------|------|
| 镜头 | ✅ | 景别+运镜组合 | "中景，跟随拍摄" |
| 主体 | ✅ | 具体描述人或物 | "穿红裙的女人，旋转" |
| 环境 | 可选 | 背景+光线+天气 | "海边，夕阳，逆光" |
| 风格 | ✅ | 视觉风格标签 | "电影感，4K，暖色调" |
| 时长 | ✅ | 具体秒数 | "5秒" |
| 转场 | 可选 | 下一镜连接方式 | "dissolve to" |

### 4.3 分镜脚本示例：《AI医生来了》情感版

```
## 分镜 1
【镜头】远景，固定
【主体】空无一人的医院走廊
【环境】深夜，荧光灯惨白，阴影浓重
【风格】悬疑电影感，低饱和度，纪实风格
【时长】4秒
【转场】fade to black

## 分镜 2
【镜头】中景，推
【主体】医生站在病床前，低头看数据
【环境】监护仪闪烁，显示心率不稳
【风格】紧张感，局部打光，阴影分割画面
【时长】6秒
【转场】cut to

## 分镜 3
【镜头】特写，跟随
【主体】AI屏幕上的诊断建议逐字出现
【环境】蓝光映射在医生脸上
【风格】科技感，数字特效，HDR
【时长】5秒
【转场】dissolve to

## 分镜 4
【镜头】全景，拉
【主体】医生后退一步，凝视镜头
【环境】走廊灯光恢复正常
【风格】情绪释放，暖色调，自然光
【时长】3秒
【转场】fade out
```

## 五、常见错误与避坑指南

| 错误类型 | 错误写法 | 正确写法 | 原因 |
|---------|---------|---------|------|
| 主体模糊 | "人走路" | "穿黑色大衣的男人在雨中街道上缓步" | AI需要具体视觉特征 |
| 缺少风格 | "一个女人在咖啡馆" | "一个女人坐在复古咖啡馆，暖色调，电影感，90年代风格" | 风格决定整体调性 |
| 运镜冲突 | "推同时拉" | "先拉后推"或选择其一 | 物理上矛盾，AI无法执行 |
| 时长过长 | "持续30秒的静态镜头" | "固定镜头8秒，慢节奏叙事" | 超长分镜易导致质量崩 |
| 转场频繁 | "每3秒一切" | "每5-8秒一个完整节拍" | 转场需要消化时间 |
| 光线矛盾 | "阳光明媚但同时阴暗" | "窗边自然光，阴影柔和" | 光线必须物理合理 |
| 忽视比例 | "全身+面部特写同框" | "用三个分镜依次展示" | 一个分镜一个重点 |

## 六、执行自检清单

写分镜提示词前，逐项检查：

- [ ] 主体描述是否具体（含动作、服装、环境）？
- [ ] 景别是否明确（远/中/近/特）？
- [ ] 运镜指令是否物理可行？
- [ ] 风格标签是否统一（cinematic/4K/HDR等）？
- [ ] 时长是否在合理范围（2-8秒）？
- [ ] 转场方式是否写清楚？
- [ ] 光线描述是否一致？
- [ ] 整体是否符合平台比例限制？
