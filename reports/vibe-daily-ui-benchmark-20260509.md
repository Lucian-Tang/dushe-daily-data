# 📱 Vibe-Daily UI 竞品分析

**分析人:** Thomas（产品经理）  
**Date:** 2026-05-09  
**目标:** 对标主流资讯产品，提出小程序 UI 改进方向

## 一、卡片设计规范

### 竞品基准

| 元素 | 36氪 | 知乎日报 | 今日头条 | 当前版本 |
|------|------|----------|----------|----------|
| 标题 | 18px Bold | 17px Medium | 18px Bold | 16px Bold ✅ 接近 |
| 摘要 | 14px 2行 | 无摘要 | 14px 3行 | 13px 2行 ✅ |
| 来源标签 | 左上角 pill | 无 | 底部 | 左上角 pill ✅ |
| 时间 | 右上角 | 底部 | 底部来源旁 | 底部独立行 ✅ |
| 配图 | 右侧 100px 方图 | 无 | 右侧大图 | **无** ❌ |
| 间距 | 12px padding | 16px | 14px | 16px ✅ |
| 圆角 | 12px | 8px | 8px | 16px ✅ |
| 背景色 | 白色卡片 | 白色 | 白色 | 暗紫渐变 ⚠️ |
| 分割 | 无 | 细线 | 无 | 卡片间距 12px ✅ |
| 操作区 | 底部 3 按钮 | 底部 2 按钮 | 底部 3 按钮 | 底部 3 按钮 ✅ |

### 差距分析
- **缺配图**：主流资讯卡片几乎都有缩略图，纯文字卡片信息密度低，识别度差
- **暗色主题**：行业主流是白底卡片，暗色卡片在白天户外可读性差
- **毒舌区分度**：当前毒舌点评和正文视觉权重相同，应降级为次级信息

### 建议
1. 如果采集时能拿到 `image` 字段，在卡片右侧展示缩略图（参考 36氪）
2. 支持日间/夜间主题切换（至少支持系统跟随）
3. 毒舌点评用更小的字号 + 左侧彩色竖线标识（当前已有，可优化对比度）

## 二、详情页规范

### 竞品基准

| 元素 | 36氪 | 知乎日报 | 当前版本 |
|------|------|----------|----------|
| 顶部导航 | 返回+分享 | 返回+点赞 | 返回+标题+分享 ✅ |
| 标题字号 | 22px Bold | 20px Bold | 22px Bold ✅ |
| 来源时间 | 标题下独立行 | 顶部 | meta 行 ✅ |
| 正文排版 | 16px, 1.8行高 | 17px, 1.7行高 | 15px, 1.7 行高 ⚠️ 略小 |
| 相关推荐 | 底部"相关文章" | 底部推荐 | **无** ❌ |
| 底部操作栏 | 固定底部：评论+收藏+分享 | 浮动按钮 | 非固定，页面内 ❌ |
| 返回手势 | 左滑返回 | 左滑返回 | 仅按钮返回 ⚠️ |

### 差距分析
- **无相关推荐**：增加用户停留时长和阅读深度的关键功能缺失
- **底部操作栏非固定**：收藏和分享按钮跟内容一起滚动，用户看完需要滚回顶部操作
- **正文字号偏小**：15px 在手机上可读性不如 16-17px

### 建议
1. 底部固定操作栏（收藏+分享+👍👎），参考 36氪
2. 正文字号改 16px
3. 底部添加 2-3 条"你可能还想看"（同频道随机推荐）

## 三、分享卡片设计（小红书风格）

### 小红书卡片特征
- **竖版** 3:4 或 9:16 比例
- **字体层级**：大标题 32-40px Bold → 正文 24-28px → 标签 18-20px
- **配色**：品牌色点缀（小红书红 #FF2442），背景以白色/米色为主
- **留白**：上下左右 40-60px，内容区呼吸感强
- **品牌元素**：底部固定位置放小程序码 + slogan
- **装饰**：顶部细线或圆点装饰，非大面积渐变

### 当前版本差距
- 背景用大面积深色渐变（暗紫→黑），和小红书"清新美图"风格不搭
- 字号偏小（标题 38px、毒舌 32px），在小红书信息流里不够醒目
- 没有"转发语"区——小红书用户习惯在分享图上有一句引导语

### 建议
1. 提供**双风格**海报：暗色"毒舌风"（现在） + 浅色"小红书风"
2. 小红书风格：白底 + 品牌色点缀 + 大标题 + 毒舌金句中号字 + 底部 slogan
3. 海报底部加"扫码看更多毒舌日报"引导文案

| 优先级 | 改进项 | 预期收益 |
|--------|--------|----------|
| **P0** | 详情页底部固定操作栏 | 收藏/分享转化率提升 30%+ |
| **P0** | 正文字号改 16px | 可读性显著改善 |
| **P1** | 日间/夜间主题 | 白天户外可用性 |
| **P1** | 浅色小红书风海报 | 分享到小红书的传播效果 |
| **P2** | 卡片缩略图 | 视觉丰富度提升 |
| **P2** | 底部相关推荐 | 用户停留时长 |
