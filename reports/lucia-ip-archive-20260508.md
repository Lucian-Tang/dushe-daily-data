## 形象基础

| 项目 | 内容 |
|------|------|
| 名称 | Lucia |
| 物种 | 🦊 狐狸（橙红色） |
| 性格 | 毒舌 / 机灵 / 傲娇 / 暖心 / 科技感 |
| 主色 | #FF6B35 → #F7931E（橙红渐变） |
| Slogan | 「我知道今天的热点在哪，但我不随便告诉你。」 |
| 画风 | Q版萌系 + 科技感，黑紫色技工外套 + choker |
| 标志元素 | 尖耳（左耳微歪）、琥珀色上挑眼、蓬松大尾巴、左手智能手环 |

## 5 种状态

| 状态 | 时段 | 姿态 | 文案 |
|------|------|------|------|
| ☀️ 晨间 | 06:00-09:00 | 半睁眼，呆毛竖，尾巴下垂 | 刚醒，别惹我 ☕ |
| 💻 工作 | 09:00-18:00 | 瞪眼，竖耳，盯着屏幕，尾巴轻晃 | 盯着一切 👀 |
| 🌙 晚间 | 18:00-23:00 | 眯眼，歪头，尾巴盘起 | 困了，明天再说 😴 |
| ⚡ 新闻触发 | 推送时 | 眼睛发光，耳朵前倾，尾巴炸开 | 重大消息！ |
| 😴 周末 | 周六日 | 四仰八叉瘫躺，流口水 | 周末摆烂中 💤 |

## 集成方式

- 顶部 Header 区显示头像 + 状态文案
- 通过 `getLuciaState()` 按时间自动切换
- 周末全天锁定周末模式
- 图片 400x400px PNG，单张 ~150KB

## 可扩展方向

1. 互动动作：点击摸头、喂食、对话
2. 节日限定皮肤（春节/中秋/圣诞）
3. 数据驱动表情（点赞多了开心，diss多了低落）
4. 语音 TTS 毒舌播报
5. 实体周边（贴纸/表情包/手办）
