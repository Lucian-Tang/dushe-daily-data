## 1. 卡片布局图（ASCII线框图）

```
┌──────────────────────────────────────────────┐
│ ┌──────┐                                     │
│ │ 角标  │  [毒舌点评标题]           ⭐⭐⭐⭐☆  │  ← 星级置右
```

### 各元素位置说明

| 元素 | 位置 | 备注 |
|------|------|------|
| 角标（平台来源） | 左上角 | 圆形/矩形，如"微博热搜""知乎" |
| 星级评分 | 右上角 | 1-5星，与标题同行 |
| 毒舌点评标题 | 标题区 | 带 emoji，如"🔥 今日最离谱" |
| 主体内容 | 中央区域 | 标题+摘要+图 |
| 警示条 | 底部，紧贴边框 | 半透明色条，左对齐文字 |
| AIGC标注 | 警示条右侧 | 灰色小字，右对齐 |

## 3. 评分浮层交互

### 点击 → 展开动画

```
用户点击星级/警示条区域
        ↓
  从触发位置向外展开（scale 0.95→1, opacity 0→1）
        ↓
  浮层高度：固定 320px，最大高度滚动
  宽度：与卡片对齐
        ↓
  浮层底部圆角 12px，与卡片底部融合
```

### 浮层内容层级

```
┌─────────────────────────────────────┐
│  ⭐ 可信度评分详情        [× 关闭]  │  ← header
├─────────────────────────────────────┤
│  ████████████░░░░  4.2 / 5.0       │  ← 进度条+分数
│                                     │
│  评估维度                            │
│  ├─ 信息来源可靠性     ★★★★☆       │  ← 分项评分
│  ├─ 内容逻辑一致性     ★★★★☆       │
│  └─ 事实核查结果       ★★★☆☆       │
├─────────────────────────────────────┤
│  💡 专家解读 / 背景说明              │
│  （2-4句补充说明）                   │
├─────────────────────────────────────┤
│  📖 相关事实核查链接 →              │  ← 跳转外链
└─────────────────────────────────────┘
```

### 关闭方式

1. **点击 × 按钮**：右上角关闭
2. **点击遮罩层**：浮层外部任意区域
3. **点击 Esc 键**：键盘关闭
4. **滑动关闭**：向下拖拽浮层超过 50% 松手后关闭

## 4. 文案规范

### 各星级警示语

| 星级 | 警示文案 |
|------|----------|
| 1星 | ⚠️ **信源待验证，请谨慎参考** |
| 2星 | 📌 **部分信息存疑，建议交叉核实** |
| 3星 | ℹ️ **信息可参考，请注意逻辑验证** |
| 4星 | ✅ **来源可靠，内容基本可信** |
| 5星 | ✔️ **信息源权威，事实核查通过** |

### AIGC标注文案

| 场景 | 文案 |
|------|------|
| 纯 AIGC 生成内容 | `🤖 AIGC 生成内容` |
| AIGC 辅助创作 | `🤖 AI 辅助` |
| 来源为 AI 报道 | `🤖 AI 消息源` |
| 待确认来源 | `❓ 来源待核实` |

> 字号：12px，颜色：`#9CA3AF`（中性灰），位于警示条最右侧

### 浮层补充文案

**1星（最高警戒）**：
> 「该内容来自非官方或匿名信源，存在未经核实的信息。建议等待权威媒体报道后再做判断。」

**2星（存疑）**：
> 「部分信息未经独立来源确认，建议多方交叉验证。关注事件官方回应。」

**3星（中立）**：
> 「信息基本可信，但部分细节仍待核实。理性看待，谨慎转发。」

**4星（较可信）**：
> 「来源可靠，核心事实有据可查。细节如有出入属正常范围。」

**5星（高可信）**：
> 「信息源自权威渠道，经事实核查机构确认。可作为参考依据。」

## 6. 与现有毒舌点评的共存方案

### 视觉分离原则

**冷静 vs 情感的对比：**

| 维度 | 可信度评分（理性） | 毒舌点评（感性） |
|------|-------------------|-----------------|
| 视觉风格 | 扁平、克制、数据化 | 活泼、有态度、带 emoji |
| 色彩系统 | 交通灯色系（绿/黄/红） | 品牌主色 + 黑色 |
| 字体 | 无衬线、规范字重 | 可用斜体、表情符号 |
| 布局 | 紧贴底部，像"注解" | 标题区，自成一行 |

### 布局共存示意

```
┌─────────────────────────────────────────────────┐
│ [角标] 🔥 今日最离谱新闻                   ⭐⭐☆ │
│                                                 │
│ 【毒舌点评】这条新闻能把你的智商气到离线，        │  ← 毒舌区（情感，主观）
│ 连标题都在挑战我的审美阈值。                      │     带语气/吐槽
├─────────────────────────────────────────────────┤
│                                                 │
│  新闻正文内容区域...                             │  ← 新闻区（客观事实）
│                                                 │
├─────────────────────────────────────────────────┤
│ ⚠️ 信源待验证，请谨慎参考          [🤖 AIGC]   │  ← 评分区（理性，冷静）
└─────────────────────────────────────────────────┘
```

### 设计隔离要点

1. **毒舌点评**：独占标题区下方，有独立背景色（如淡紫 `#F5F3FF` 或透明），正文用斜体/活泼字体
2. **可信度评分**：固定底部条，像 footer stamp，不抢主体视觉
3. **间距**：毒舌区与评分区之间保持 12px 间距，形成明确分区
4. **交互独立**：毒舌区点击可展开长文；评分区点击展开浮层 —— 互不干扰

### 配色对比示例

```
毒舌区背景：  #F5F3FF  淡紫（活泼感）
评分区背景：  #FEE2E2  淡红 / #DCFCE7 淡绿（功能性）
```

## 附录：技术实现备注

1. 浮层建议用 `position: absolute` + `transform-origin: top center`，动画 200ms ease-out
2. 警示条固定高度 36px，内容超长截断 + ellipsis
3. AIGC 标注在无评分数据时也显示（独立行）
4. 星级使用 SVG 星星，filled/empty 两种状态，避免 emoji 兼容性问题
5. 浮层内分项评分为可选项，数据不足时降级显示总分
