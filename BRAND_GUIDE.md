## 品牌标识

### Logo 概念

**主图形：** 灯塔+棱镜 —— 一束光穿过三棱镜，折射出三道光芒（确定性/决策力/品味）

**文字标识：** Luxmind（首字母 L 大写，其余小写）

### 色彩体系

| 用途 | 色值 | 名称 |
|------|------|------|
| 品牌主色 | `#FFD700` | Lux Gold（光） |
| 背景深色 | `#0F0F1A` | Abyss Black（深渊） |
| 背景中层 | `#1A1A2E` | Midnight Blue（午夜） |
| 确定性 | `#00D4AA` | Truth Green（真实绿） |
| 决策力 | `#4A90D9` | Action Blue（行动蓝） |
| 品味 | `#E85D75` | Taste Red（品味红） |
| 文字主色 | `#E8E8E8` | Light Gray |
| 文字次色 | `#999999` | Muted Gray |

### 字体

- **中文：** 思源黑体 / PingFang SC
- **英文：** Inter / SF Pro Display
- **数字：** Tabular Figures（DIN 风格）

### 品牌声音

**语气：** 直接、有态度、不端着  
**原则：** 毒舌外壳+诚实内核  
**不用词：** "领先的""一流的""致力于""赋能"  
**常用词：** "说实话""别看""值得花时间""我们验证过了"
